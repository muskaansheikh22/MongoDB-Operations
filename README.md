HOW TO RUN
==========
 
1. Open PowerShell.
 
2. Go to the folder with the file:
   cd path\to\mongo_project
 
3. Run it:
   mongosh mongoproject.js
 
That's it. You should see the employees, inventory, payments, and promo
data printed out.
 
If "mongosh" is not recognized, MongoDB / mongosh is not installed or
not on your PATH yet.
